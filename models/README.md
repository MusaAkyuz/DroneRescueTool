# YOLO Model Dizini

Bu dizine eğitilmiş YOLO model dosyanızı (`best.pt`) yerleştirin.

## Kullanım

1. YOLO modelinizi (`best.pt`) bu dizine kopyalayın.
2. Uygulama başlatıldığında model otomatik olarak yüklenecektir.

## Desteklenen Formatlar

- `best.pt` — YOLOv8/YOLOv11 PyTorch model ağırlıkları
